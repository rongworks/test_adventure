# TestAdventure

A simple DSL for writing test-plans for explorative testing.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'test_adventure'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install test_adventure
